# Weather App

### Description:

<p style="color: red;">
  Weather app created using NextJS, React, and Typescript.
  <br>
  Select a location on the map then press the button to receive the current weather data.
</p>

---

### Features:

- [x] React + Typescript
- [x] NextJS
- [x] Linting (ESLint)
- [x] Formatting (Prettier)
- [x] Testing (Jest)
- [ ] Containerized (Docker)
